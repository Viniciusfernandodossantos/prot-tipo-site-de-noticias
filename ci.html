1. Como criar uma conta no GitHub?

  1. Acesse o site do GitHub: Vá para github.com.
  2. Clique em "Sign up" (Inscrever-se).
  3. Preencha o formulário de inscrição:
    • Username: Escolha um nome de usuário único.
    • Email: Insira seu endereço de e-mail.
    • Password: Crie uma senha forte.
  4. Verifique seu e-mail: O GitHub enviará um e-mail para o endereço fornecido. Abra o e-mail e clique no link de verificação.
  5. Complete a configuração: Após a verificação, você pode ser solicitado a configurar algumas preferências iniciais, como a escolha de um plano (o plano gratuito é suficiente para a maioria dos usuários iniciantes).

2. Quais são os primeiros passos após a criação da conta no GitHub?

  1. Complete seu perfil:
    • Vá para as configurações do perfil e adicione uma foto e uma breve descrição para tornar seu perfil mais completo.
  2. Crie um novo repositório:
    • Clique no botão "New repository" (Novo repositório) na sua página inicial do GitHub.
    • Dê um nome ao repositório, escolha se será público ou privado e, opcionalmente, adicione um arquivo README.
  3. Explore e siga outros repositórios e desenvolvedores:
    • Navegue por projetos e siga desenvolvedores para ficar atualizado sobre novos projetos e boas práticas.
  4. Configure as notificações:
    • Ajuste suas configurações de notificação para não perder atualizações importantes de seus repositórios e projetos que você está seguindo.

3. Como configurar o Git no seu computador local para se conectar ao GitHub?

  1. Instale o Git:
    • Windows: Baixe o instalador do site oficial do Git e siga as instruções.
    • Mac: Você pode instalar o Git usando Homebrew com o comando brew install git, ou baixar o instalador do site oficial.
    • Linux: Use o gerenciador de pacotes da sua distribuição, por exemplo, sudo apt-get install git para distribuições baseadas em Debian/Ubuntu.
  2. Configure suas informações de usuário: 
    
git config --global user.name "Seu Nome"
git config --global user.email "seuemail@example.com"
  3. Gerar uma chave SSH: 
    
ssh-keygen -t rsa -b 4096 -C "seuemail@example.com"
    • Pressione Enter para aceitar o local padrão e defina uma senha, se desejar.
    • Após gerar a chave, adicione a chave SSH ao seu agente SSH: 
      
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_rsa
  4. Adicione a chave SSH ao seu GitHub:
    • Copie a chave SSH para a área de transferência: 
      
cat ~/.ssh/id_rsa.pub
    • No GitHub, vá para Settings (Configurações) > SSH and GPG keys > New SSH key.
    • Cole a chave copiada e salve.
  5. Clone um repositório para testar: 
    
git clone git@github.com:usuario/nome-do-repositorio.git
